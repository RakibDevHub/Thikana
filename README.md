# 🏠 Thikana Real Estate Platform

A full-stack real estate platform where admins can list properties and users can browse without login.

## 📋 Features

- **Property Listings** - Browse available properties
- **Search & Filter** - Find properties by location, price, type
- **Property Details** - View full property information
- **Contact Form** - Inquire about properties (no login required)
- **Testimonials** - Client reviews with moderation
- **Admin Panel** - Manage properties, testimonials, inquiries

## 🛠️ Tech Stack

- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Styling**: CSS
- **Database**: (Coming soon)
